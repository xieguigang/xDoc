# TaskManager
_namespace: [Microsoft.VisualBasic.Win32](./index.md)_

Windows的任务管理器的接口



### Methods

#### ProcessUsage
```csharp
Microsoft.VisualBasic.Win32.TaskManager.ProcessUsage
```

#### ProcessUsageDetails
```csharp
Microsoft.VisualBasic.Win32.TaskManager.ProcessUsageDetails
```

_returns: Memory, CPU_
