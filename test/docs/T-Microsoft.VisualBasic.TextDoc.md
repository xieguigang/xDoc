### Methods

#### IsTextFile
Determined that the target file is a text file or binary file?
 (判断是否是文本文件)

_returns: 是返回True，不是返回False_
#### IterateAllLines
通过具有缓存的流对象读取文本数据，使用迭代器来读取文件之中的所有的行，大文件推荐使用这个方法进行读取操作

_returns: _
#### ReadAllLines
这个函数只建议读取小文本文件的时候使用

_returns: _
#### ReadAllText
这个函数只建议读取小文本文件的时候使用

_returns: _

_returns: _
