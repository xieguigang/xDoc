
# PathMapper
_namespace: [Microsoft.VisualBasic.Language.UnixBash](N-Microsoft.VisualBasic.Language.UnixBash.md)_

这个模块是将Linux路径映射为Windows路径的

### Methods

#### GetMapPath
Map linux path on Windows:
 [~ -> C:\User\<user_name>]
 [# -> ]
 [/ -> C:\]
 [/usr/bin -> C:\Program Files\]
 [/usr -> C:\User\]
 [- -> ]
_returns: _
#### HOME
Get user home folder
_returns: _
